1) Login and Signup View

Sign in and Signup pages are an important page as this allows the user to login to the website system and start collecting the information required.

You can view the pages for both Login and Signup

2) Homepage view

During the contextual analysis most of the users were unclear about the various literature types available in the homepage and hence were unable to pick the appropriate choices even when they were available. Having these in mind the homepage was designed with the literature options embedded in the menus that would present the choices to users in an effective way.

Features

Here is a list of features available on Homepage.

  • General Introduction
  • Direct links to Journals, Thesis, Magazines and ACM Books
  • Top 10 Subject Areas with Filters (today,this week, this month, this year)
    • Total Papers in the subject area are listed
    • Total views are listed
  • Top 5 Research Papers with Filters (today,this week, this month, this year)
    • Quick View to view a paper is available
    • Quick bookmark to refer later is available
  • Fully Responsive Pages

3) Search Bar/Page view

During our contextual analysis, we found that most users to navigate throughout the website, used search bar. It is one of the most prominent feature for the ACM library and must be made easy for the user to search while providing best results

Features
Search Bar
  • Ajax Search (try typing Jigsaw, Google, journal,...)
  • Direct links to Journals, Thesis, Magazines and ACM Books
Search Page
  • Enter search term with Advanced Filters
  • Displays results with recommended feature
  • Displays results with recommended papers
  • Quick view, Bookmark, Download and remove from search results available
  • Quick view, Bookmark, Download and remove from search results available
  • Realted Searches with searched keyword is provided
  • Top 10 results, matching the keyword density are provided on the first page.

4) Journals Listing view

The list of various Journals, conferences, proceedings was vast and hence an effective way to list and search the list of journals.

Features
  • Journal Search Available (no functionality added for this prototype)
  • Total downloads, citations and years are available
  • Popular Subject area is given to reduce user's work with one click

5) Conference Listing view

The list of various Journals, conferences, proceedings was vast and hence an effective way to list and search the list of journals.

Features
  • Journal Search Available (no functionality added for this prototype)
  • Total downloads, citations and years are available
  • Popular Subject area is given to reduce user's work with one click
  • Conference Search Available (no functionality added for this prototype)
  • Individual Conference Listing Provided view
  • Individual Conference Subject Area and Advanced Filters view

6) MyCollection view

A new feature that we wanted to add to our website prototype. This will help user to navigate to their respective bookmarked papers and submissions with ease

Features
  • Can view demographics and content with time(All time, Yearly, weekly, Daily) filters.
  • Demographic overview is given for bookmarks, total searches and total submission
  • Bookmarks with filters ( with quick view and remove bookmark features)
  • List of submissions
  • List of search queries (with re-requery and remove search options)

7) Profile view

A new feature that we wanted to add to our website prototype. One stop solution for users to change their account and notification settings.

Features
  • User can change account information
  • User can update Notification Settings
